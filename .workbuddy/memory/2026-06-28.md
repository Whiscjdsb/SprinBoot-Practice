## 关键概念
- AOP 代理机制：Spring 为 Controller/Service 创建代理对象，外部调用实际走代理
- @Around vs @Before：@Around 包裹整个方法调用，可控制前后逻辑并获取返回值
- joinPoint.getTarget() 获取被拦截的目标对象，joinPoint.getSignature() 获取方法签名
- 切点表达式：controller..*.* 匹配包及其子包，controller.*.* 只匹配直接子类
- try-finally 陷阱：finally 中 return 会覆盖 catch 中的异常，应与 try-with-resources 结合

- Java 基础需要补充（Object、继承、接口、异常体系）
- MVC 三层架构：Model（数据）→ Controller（调度）→ View（展示）
