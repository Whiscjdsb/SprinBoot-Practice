## Docker 部署完成
- 创建 Dockerfile（多阶段构建：Maven build + JRE run）
- 创建 docker-compose.yml（mysql + redis + app 三个服务）
- 配置国内镜像加速器（docker.1ms.run, docker.xuanyuan.me）
- **关键 bug**：项目同时有 application.properties 和 application.yml，.properties 优先级更高，导致 yml 里的 ${DB_HOST:localhost} 环境变量不生效，app 一直连 localhost 而非 mysql 容器
  - 修复：把 application.properties 改成环境变量版本，删除 application.yml
- 添加 healthcheck + condition: service_healthy 确保 MySQL 就绪后 app 才启动
- 创建 init.sql 挂载到 /docker-entrypoint-initdb.d/，MySQL 首次启动自动建表+插入测试数据
- 最终 `docker compose up -d` 一键启动成功，定时任务正常查询用户数据
