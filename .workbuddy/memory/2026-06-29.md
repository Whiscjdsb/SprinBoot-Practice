## Phase 4：Spring Security + JWT 完成

### 新增依赖
- spring-boot-starter-security
- jjwt-api, jjwt-impl, jjwt-jackson (0.12.5)

### 新增文件
- security/SecurityConfig.java — 安全配置，所有接口需认证，/api/auth/** 放行
- security/JwtUtil.java — JWT 生成、验证、解析工具
- security/JwtAuthenticationFilter.java — 每次请求检查 Header 的 Bearer token
- controller/AuthController.java — POST /api/auth/login 和 /api/auth/register
- dto/LoginRequest.java — 登录请求 DTO

### 修改文件
- UserService.java — 新增 findByUsername(String username) 方法

### 验证结果
- 不带 token 访问 /api/users → 403
- 注册 → 返回 token
- 带上 token（Authorization: Bearer xxx）访问 /api/users → 200 成功

### 用户偏好
- 通用配置代码（SecurityConfig、JwtUtil、JwtAuthenticationFilter）可以复制粘贴，不需要背诵
- 业务逻辑（Controller、Service 方法）让学生自己写
- 每一行都要解释什么意思
