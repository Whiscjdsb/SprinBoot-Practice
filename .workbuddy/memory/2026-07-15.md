### 核心概念
- bootstrap.properties 先于 application.properties 加载（连 Nacos 拿远程配置）
- @RefreshScope: Nacos 配置变了，Spring 自动重建 Bean，不重启生效
- @Value("${key:默认值}"): 读配置，冒号后是兜底值
- Data ID 命名: ${spring.application.name}.${file-extension}
- spring.config.import: Spring Boot 3.x 必须显式声明配置导入来源

### 踩坑
- bootstrap.properties 在 Spring Boot 3.x 默认不加载 → 配置写在 application.properties
- spring.config.import=nacos: 后面必须带 dataId（不能只写 nacos:）
- spring.config.import=optional:nacos:notification-service.properties ← 正确格式
- IDEA 改了 application.properties 不自动更新 target/classes → 需要 Build → Rebuild Project 或 Maven clean compile
- Spring Cloud Alibaba 2023.0.1.0 强制要求 spring.config.import，不加会启动失败
